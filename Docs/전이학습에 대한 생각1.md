## KT AIVLE School 미니 프로젝트 진행간에 발생한 생각정리 

### 1. 전이학습의 관점 

전이학습을 사용하는 관점은 

1. 데이터셋이 작고 이미지넷과 유형이 유사
2. 데이터셋이 크고 이미지넷과 유형이 유사
3. 데이터셋이 작고 이미지넷과 유형이 상이
4. 데이터셋이 크고 이미지넷과 유형이 상이


### 2. 미니프로젝트에서 사용한 데이터의 형태

- 이미지가 600장 정도로 데이터의 크기가 매우 작음 
- 차량 파손/ 정상 2가지 데이터인데 결론적으로 특징을 뽑아내는 것 
- 따라서 위에 관점 4가지 중에서 1번 유형으로 생각

### 3. 유형에 따른 전이학습의 관점 

- 1번 유형의 경우 전이학습으로 불러오는 "특징 추출" 부분은 고정하고 뒤에 작성하는 분류기 부분만 학습을 진행하는 경우 일반적인 성능이 좋다고 알고 있었다.

- 그러나 이후에 "특징 추출" 부분은 미세하게 건드리며 결과의 변화를 확인하는 것도 고려해봄직한 방법인것 같다.


### 4. 결론

- 미프에서 사용한 데이터셋은 크기가 작은 편이라 특징을 뽑아내기에는 양이 부족하다고  볼수 있는데 그래서 모델 전체를 학습하게 되면 과적합이 발생하는 것 같다.

- 이 과정에서 어떤 현상이 일어나는지를 알아두는 것이 도움이 될 것 같다.
